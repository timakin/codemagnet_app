How it works

What is this all about?

CodeMagnet is code-curated platform. It's a place for hackers who wanna find latest code trend, beautiful code, best practices. Enthusiastic hackers can share own code, or new update of a favorite programming language.

What's the goal of CodeMagnet?

The goal of CodeMagnet project is to improve the method of code reading and make renaissance of programming learning with enabling everyone to access to awesome code in casual.

May I submit a code?

Sure! But each code should include the:

・Title: simply summarized description about code.
・Url: Direct link to code (avoid unreadable code, like whole repository)
・Language: With which language the code is written
・Introduction: (How to use, Why is the code awesome, the logical relationship with other methods or modules.)

Examples:

・Title: High speed search with ag
・Language: C
・Introduction:

The substitute of grep command, "ag" is super quick to search. BM algorithm is the reason. Boyer-Moore String Search Algorithm is an efficient string searching. The key feature of the algorithm is to match on the tail of the pattern rather than the head, and to skip along the text in jumps of multiple characters rather than searching every single character in the text.
