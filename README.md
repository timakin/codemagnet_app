# CodeMagnetApp

## Info
- Written in `ReactJS/AngularJS` and `NodeJS`
- build by `grunt`
- test by `mocha/JSHint`


## How to develop
### instalation
- `npm i && bower install`

### daemon preparation
- `mongod --dbpath /path/to/your/datadir &` # depends on your environment, anyway, you need mongo.
- `grunt serve`

### build & test
- `grunt`
